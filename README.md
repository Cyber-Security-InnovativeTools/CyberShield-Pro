
# 🛡️ CyberShield Pro - Website Pentesting & Security Scanner

CyberShield Pro is a powerful web-based security scanner and traffic analyzer built using Python Flask.

## 🚀 Features

* 🔍 Full Website Security Scan
* ⚡ Quick Scan (HTTP + SSL)
* 📊 Real-time Traffic Analytics
* 🧠 Risk Score Calculation
* 🔐 Security Headers Check
* 🌐 DNS & SSL Analysis
* 📈 Beautiful Dashboard UI

## 🛠️ Tech Stack

* Python (Flask)
* SQLite Database
* Pandas & NumPy
* Matplotlib & Plotly
* Bootstrap UI

## ⚙️ Installation

```bash
git clone https://github.com/your-username/cybershield-pro.git
cd cybershield-pro
pip install -r requirements.txt
python app.py
```

## 🌍 Usage

Open in browser:

```
http://localhost:5000
```
